TSTerm home page

The test proxy server also serves static pages to test in a browser. Navigate to the following URL in a modern browser:

  • http://localhost:proxyPort/static/app/termtest.html?address=host&port=port
  • For TN3270 over TLS: http://localhost:proxyPort/static/app/termtest.html?address=host&port=port&useTLS=true
